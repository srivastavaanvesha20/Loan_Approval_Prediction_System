# Loan Approval Prediction System

An end-to-end Machine Learning project that predicts loan approval status using applicant financial and credit information.

## Features
- Data Preprocessing
- Feature Engineering
- Model Comparison
- XGBoost Classifier
- Streamlit Dashboard

## Models Used
- Logistic Regression
- Random Forest
- XGBoost

## Best Model
XGBoost Classifier

Accuracy: 98.36%

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- XGBoost
- Streamlit
